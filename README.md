# 🌐 Wireshark Network Traffic Analysis Project

## 📌 Overview

I captured and analyzed live network traffic using Wireshark to understand protocol behavior and identify potential security threats.

This project simulates real-world network monitoring and SOC analysis.

---

## 🎯 Objectives

* Capture live network traffic
* Analyze key protocols
* Identify normal vs suspicious behavior
* Apply filtering techniques
* Document findings

---

## 🛠️ Tools Used

* Wireshark

---

## 🔍 Protocols Analyzed

### 🌐 DNS

* Domain name resolution
* Query and response analysis

### 🔗 TCP

* Three-way handshake
* Session establishment

### 📡 ICMP

* Network connectivity testing
* Echo request/reply

### 🔐 HTTPS

* Encrypted communication
* Secure web traffic

---

## 🧪 Methodology

1. Captured live traffic over Wi-Fi
2. Applied filters to isolate protocols
3. Analyzed packet details
4. Followed TCP streams
5. Identified patterns and anomalies

---

## 🔎 Key Filters Used

```txt
dns
tcp
icmp
http
tls
```

---

## 🚨 Observations

### Normal Activity

* DNS queries resolving domains
* Successful TCP handshakes
* ICMP responses confirming connectivity
* HTTPS encrypted sessions

---

### Suspicious Indicators

* Repeated DNS queries
* Multiple connection attempts
* Unusual traffic patterns

---

## 📈 Key Takeaways

* Deep understanding of network protocols
* Ability to analyze packet-level data
* Identification of suspicious behavior
* Hands-on experience with Wireshark
